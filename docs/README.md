## Domain Kararı

Birincil domain:

> https://www.matematik-akademi.com/

SEO ve canonical URL yapısı bu domain üzerinden hazırlanacak. Kök domain olan `https://matematik-akademi.com/` yayında `www` adresine yönlendirilmelidir.
